# Application Patterns Canon

## Allowed Patterns
- Aggregate Root
- Value Object
- Factory Method
- Domain Event
- Command / Query + Handler
- Policy Evaluator
- Strategy
- Adapter
- Specification (limited)
- Outbox (future)

## Forbidden Patterns
- Generic Repository
- Manager classes
- God Services
- Anemic Domain Models
- Service Locator
- Reflection-heavy magic

## Rule of Gold

Behavior first.
Pattern second.
Never pattern before problem.
