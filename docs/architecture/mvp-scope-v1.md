# AIX MVP Scope v1.0

## Objective

Deliver a configurable governance-oriented DMS/ECM platform for SMEs.

## AI / OCR stance

OCR and AI are optional future capability layers. **AIX v1.0 must be AI-ready but not AI-dependent.**

v1.0 does not include advanced AI runtime, semantic search, or advanced OCR pipelines.

## Included

- Platform Auth
- Tenant Registry
- Angular 21 + PrimeNG + Avalon
- declarative renderer
- versioned DocumentTypes
- reusable Keywords
- taxonomy
- COMPLETE state
- versionable documents
- declarative policies
- REST + OpenAPI
- webhooks
- provider abstraction
- behavior-first testing

## Excluded

- advanced AI runtime
- semantic search
- advanced OCR
- BPM/workflow engine
- ERP integrations
- SSO/passkeys
- microservices
