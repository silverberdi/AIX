# Ubiquitous Language

## Core Concepts

### Keyword
Atomic field definition.

### Group
Reusable collection of keywords.

### Table
Repeating tabular structure.

### Document Type
Composable document definition.

### Document Instance
Captured document data.

### Rule
Declarative conditional behavior.

### Renderer
Dynamic execution engine for forms.

### Capture Mode
UPLOAD, SCAN, or EFORM.
