# Expedient Canon

## Definition
An expedient is an operational document container. It is not a document.

## Principle
Expedient = operational grouping unit.
Document = evidence unit.

## Responsibilities
- group documents
- provide operational context
- support navigation
- organize related evidence
- support cases and business processes

## Non-Responsibilities
An expedient does not replace documents, document identity, document metadata, document audit, or document retention policies.

## MVP Capabilities
- create expedient
- close expedient
- associate documents
- consult associated documents
- define basic expedient metadata
