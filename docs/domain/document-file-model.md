# AIX Document File Model

## Principle
A document may contain one or multiple physical files.

## File Categories
### Primary File
The main evidence artifact of the document.

### Supporting Files
Complementary files: attachments, receipts, images, DIAN responses, tickets, scanned evidence.

### Derived Files
System-generated files: thumbnails, previews, OCR text, generated PDFs, conversions.

## Configuration
The file structure is configurable by DocumentType:
- primary file required or optional
- allowed file types
- required supporting files
- optional supporting files
- OCR requirement
- generated files

## Rule
The file composition of a document is configurable by DocumentType.
