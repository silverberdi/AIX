# AIX Product Roadmap

## Product Positioning

AIX is a governance-oriented modular DMS/ECM platform for SMEs.

## Product Principles

1. Governance over storage.
2. Metadata over folders.
3. Policies over hardcoded behavior.
4. Search over navigation trees.
5. Workspaces over dashboards.
6. Contracts over implicit integrations.
7. Modularity over fragmentation.
8. AI-ready, not AI-dependent.
9. Tenant isolation by design.
10. Operational simplicity over enterprise complexity.
11. Declarative UX over hardcoded forms.
12. Evolutionary architecture over premature microservices.

## Phases

### Phase 0 — Foundation
- modular monolith
- multitenancy
- platform/business separation
- document aggregate
- metadata system
- renderer
- uploads
- governance MVP
- search MVP
- security MVP

### Phase 1 — Operational DMS
- capture optimization
- bulk operations
- advanced search
- queues/workspaces
- datasets/lookups

### Phase 2 — Governance Expansion
- retention execution
- legal hold
- disposition workflows
- compliance reporting

### Phase 3 — Integration Platform
- mature external APIs
- advanced webhooks
- ERP integrations
- orchestration

### Phase 4 — AI Foundation
- OCR pipelines
- extraction suggestions
- embeddings
- semantic enrichment

### Phase 5 — Intelligent Governance
- anomaly detection
- semantic linking
- intelligent classification
- AI-assisted audits
